# tv-shows-application


## Installations
## Install Vue CLI 

```
npm install -g @vue/cli
```

## Project setup
```
npm install
```

### Compiles for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
